# Day 4: Tuples and Data Processing

## Overview

This folder contains the code for Day 4, featuring multiple return values, tuple creation, and more complex dictionary operations.

## Key Concepts Covered

- **Multiple Return Values**: Returning multiple values (min, max, sum) from a single function using tuples.
- **String Cleaning**: Processing a list of strings to remove whitespace and convert to lowercase.
- **Tuples**: Creating lists of tuples to store paired data (e.g., word and its length).
- **List Comprehensions**: Using concise syntax to create lists (an alternative to loops).
- **Dictionary Creation**: Manually merging two lists (keys and values) into a dictionary.
- **Data Transformation**: Filtering and modifying data in a single pass (e.g., squaring odd numbers).
- **Categorization**: Grouping data into categories (pass/fail) using a dictionary of lists.

## Code Files

- `day4.py`: Solutions for `multi_func`, `clean_str`, `list_tuple`, `list_merger`, `filt_func`, and `categorize`.
